### Set Environment Variables

Rename the .envexample to .env and add your [MongoDB](https://www.mongodb.com/) database URI and your JWT secret

### Install backend dependencies

```bash
cd server
npm install
```

### Install client dependencies

```bash
cd client
npm install
```
